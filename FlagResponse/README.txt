Installation
------------

Unzip the files into the Tribes\config dir.  No other installation is necessary,
AutoLoad will find and load the script.

Now start tribes, go to the Options menu, then the Scripts menu, now select
Flag Response from the pull down menu.  From here you can set various options
and get help.

Advanced: The file FlagResponse.avl.vol can be located anywhere, the file 
FlagResponseMessages.cs must be in the config dir.
